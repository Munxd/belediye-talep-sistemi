# Belediye Talep ve Şikâyet Sistemi

Vatandaşların talep ve şikâyetlerini çevrim içi olarak iletebildiği, ilgili belediye birimlerinin kayıtları takip edip sonuçlandırabildiği web tabanlı yönetim sistemidir.

## Proje Amaçları

- Vatandaşların talep ve şikâyet oluşturabilmesi
- Başvuruların ilgili müdürlüklere yönlendirilmesi
- Talep durumlarının takip edilmesi
- Yönetici paneli üzerinden rapor ve istatistik sunulması
- Yapay zekâ ile kategori ve öncelik önerisi yapılması

## Kullanılacak Teknolojiler

- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Bootstrap
- Chart.js
- Yapay zekâ destekli metin analizi

## Kullanıcı Rolleri

- Vatandaş
- Personel
- Yönetici
